# Symptom AI Desease Detection

Hybrid Machine Learning and Generative AI system for disease prediction, symptom understanding, explainable medical reasoning, and personalized prevention recommendations.

## Features

- Natural language symptom input
- LLM-based symptom extraction
- Interactive symptom validation
- Disease prediction using Machine Learning
- AI-powered medical reasoning
- Prevention and recommendation generation
- Streamlit web application

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Random Forest
- Large Language Models (LLM)
